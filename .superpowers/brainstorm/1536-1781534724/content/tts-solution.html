文字转语音（TTS）方案

AI 女友说话的声音，直接影响沉浸感和情感体验

A

商业 TTS API（高质量）

代表：OpenAI TTS、Azure TTS、ElevenLabs

优势

  • 音质最好，接近真人
  • 支持情感表达（OpenAI、ElevenLabs）
  • 延迟低（1-2秒）
  • 多语言支持
  • 集成简单

劣势

  • 成本较高（$15/1M 字符左右）
  • 需要网络请求
  • 声音选择受限于预设

推荐：OpenAI TTS（$15/1M字符）或 ElevenLabs（音质最好但贵）

B

国内 TTS API（便宜稳定）

代表：阿里云、腾讯云、科大讯飞、火山引擎

优势

  • 价格便宜（约 $0.2-1/1M 字符）
  • 国内访问稳定
  • 中文发音质量好
  • 部分支持情感标注（讯飞）

劣势

  • 音质略逊于国际一线
  • 情感表现力一般
  • 有些声音偏"播音腔"

推荐：火山引擎（字节，音质较好）或阿里云（便宜）

C

本地开源 TTS

代表：VITS、GPT-SoVITS、Bert-VITS2、Piper

优势

  • 完全免费，无调用限制
  • 可以训练专属音色（声音克隆）
  • 数据私有
  • 音质可以很好（GPT-SoVITS）

劣势

  • 需要 GPU 推理（但 CPU 也能跑，只是慢）
  • 训练音色需要素材和技术
  • 部署复杂
  • 推理速度慢（本地限制）

推荐：GPT-SoVITS（音质好）或 Piper（轻量快速）

D

语音克隆 API

代表：ElevenLabs Voice Cloning、Azure Custom Neural Voice

优势

  • 可以用任意声音（声优、明星等）
  • 只需 1-5 分钟音频样本
  • 音质和表现力都很好
  • 个性化最强

劣势

  • 成本最高（ElevenLabs $11-99/月订阅）
  • 有版权和伦理风险
  • 部分服务需要声音所有者授权

推荐：ElevenLabs Professional（克隆质量最好）

我的推荐：B（国内 TTS API）

理由：

  • 你用 Deepseek（国内），TTS 也用国内更统一
  • 火山引擎的音质已经很不错，能满足 80% 需求
  • 成本低，每天聊天消耗不到 $0.1
  • 国内访问稳定

升级路径：如果未来需要更强情感表现，可以：

  • 方案 1：上 OpenAI TTS（情感表达更好）
  • 方案 2：自己部署 GPT-SoVITS，训练专属音色（长期运营值得）

成本对比：

  • 国内 API：约 $0.05-0.1/天/用户
  • OpenAI TTS：约 $0.3-0.5/天/用户
  • 本地方案：免费，但需要服务器/GPU
